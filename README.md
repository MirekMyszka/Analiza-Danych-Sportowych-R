# Analiza Danych Piłkarskich: Elo, Momentum i Rynki Bukmacherskie

## Opis projektu
Projekt zawiera kompleksową analizę statystyczną meczów piłkarskich z lat 2000-2025. Badanie skupia się na trzech filarach: sile zespołów (ranking Elo), aktualnej formie (momentum) oraz kursach bukmacherskich.

## Główne pytania badawcze:
* Jak ranking Elo koreluję z rzeczywistym wynikiem meczu?
* Czy "mecze na szczycie" są trudniejsze do przewidzenia?
* Które ligi światowe cechują się największą nieprzewidywalnością?
* Czy "dobra seria" faktycznie gwarantuje punkty w kolejnym meczu?

## Technologie
* **Język:** R
* **Biblioteki:** `tidyverse`, `ggplot2` (zaawansowane wizualizacje danych)
